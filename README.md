# pydiffold


## Features
* Compute normals
* Compute tangent bundle
* Compute normal bundle
* Compute metric tensor
* Compute geodesics
* Define functions in manifolds
* Compute gradient
* Compute surface gradient
* Compute Laplace-Beltrami
